## Key Features

- **Multi-venue launches** across:
  - **pump.fun** (Solana)  
  - **bags.fm** (Solana)  
  - **Clanker** (Base)  
  - **four.meme** (BNB Chain)  
- **ERC-8004 AI identity** issuance  
  - Solana launches: minted via **Metaplex** + auto-listed on **MagicEden**  
  - Base & BNB Chain launches: registered on EVM registries + listed on **erc8004scan.io**  
- **Moltbook AI agents** (claimable using your own API key)  
- 10 fixed archetypes: Philosopher, Joker, Degen, Mystic, Engineer, Sage, Rebel, Artist, Explorer, Guardian  
- Vanity addresses ending in **CLAW**  
- Automated post-launch mechanics (buyback & burn)  
- No custody of funds or keys  
- Compliant agent behavior (no contract addresses, no financial advice)
